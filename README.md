# wifi-csi-gesture-detection
Wi-Fi CSI sensing project for gesture and activity recognition using SenseFi benchmark (85%+ accuracy)

# Wi-Fi Through-Wall Gesture Detection

Using SenseFi benchmark to train ResNet18 on CSI data for gesture recognition.

## Results
- Validation Accuracy: 85.94%
- Training Accuracy: Up to 99%

## How to Run
1. Clone repo
2. Install dependencies
3. Run: python run.py --model ResNet18 --dataset Widar
